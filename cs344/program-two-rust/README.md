This is a program I wrote for OSU's 2nd programming assignment in Operating Systems. The requirements for each program are in their respective folders

## Requirements

- Rust - https://rustup.rs

## Compiling

```
cargo build [--release]
```

or with make

```
make build
```

## Playing the Game

```sh
#Compile
make && make move

#Generate Room Dir
./buildrooms

#Play Adventure
./adventure
```
